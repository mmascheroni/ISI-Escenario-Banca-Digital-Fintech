# 07 — Riesgos Residuales

| Riesgo | Justificación |
|---|---|
| Phishing avanzado | MFA reduce impacto pero no elimina ingeniería social |
| Malware móvil | Root/jailbreak puede comprometer dispositivo |
| Insider threat | Empleados privilegiados siguen representando riesgo |
| Dependencia terceros | Caída de pasarela pagos afecta disponibilidad |
| Zero-day vulnerabilities | Riesgo inherente no totalmente mitigable |
